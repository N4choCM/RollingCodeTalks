Speakers

Clean code
Jhonny Guzmán

Clean Code, o Código Limpio, es una filosofía de desarrollo de software que consiste en aplicar técnicas simples que facilitan la escritura y lectura de un código, volviéndolo más fácil de entender. Es decir, el clean code es una buena práctica.

CSS
Pablo Frías

CSS son las siglas en inglés para «hojas de estilo en cascada» (Cascading Style Sheets). Básicamente, es un lenguaje que maneja el diseño y presentación de las páginas web, es decir, cómo lucen cuando un usuario las visita.

Backend
Walter Rivas

El backend es la parte del desarrollo web que se encarga de que toda la lógica de una página web funcione. Se trata del conjunto de acciones que pasan en una web pero que no vemos como, por ejemplo, la comunicación con el servidor.

Ciberseguridad
Barack Obama

La ciberseguridad es la práctica de proteger sistemas, redes y programas de ataques digitales. Por lo general, estos ciberataques apuntan a acceder, modificar o destruir la información confidencial o interrumpir la continuidad del negocio.

Machine Learning
Nina Müller

El Machine Learning es una disciplina del campo de la Inteligencia Artificial que, a través de algoritmos, dota a los ordenadores de la capacidad de identificar patrones en datos masivos y elaborar predicciones (análisis predictivo).

Big Data
Lionel Messi

Big Data es un término que describe el gran volumen de datos, tanto estructurados como no estructurados, que inundan los negocios cada día. Estos datos se pueden analizar para obtener ideas que conduzcan a mejores decisiones de negocios.

SITE CENTRO - Octubre 2022

Gral. Paz 576 Piso 9 – SMT

En RollingCode somos un equipo de profesionales apasionados y comprometidos. Tenemos nuestra Factory, donde trabajamos desarrollando apps y sitios webs para nuestros clientes. También contamos con un Labs, donde creamos nuestros propios software: investigamos productos, tecnologías y ponemos en marcha MVPs para probarlos en el mercado y que luego puedan crecer por sí mismos. Las puertas están siempre abiertas para quienes comparten nuestro propósito de generar un cambio en sus vidas, quienes quieren crecer y desarrollarse en un entorno de compañerismo y profesionalismo.
